## Download link

https://www.anaconda.com/download/

## Activate installation

```bash
source ~/.bashrc
```

## Test installation

```bash
conda list
```

## Create venv

```bash
conda create --name my_env python=3
```

## Activate the new environment like so:

```bash
source activate my_env
```
