# Commands

### Open file

```bash
vim file.txt
```

### Insert mode

    i

### Exit insert mode

    ESC
    
### Scroll up and down

    Arrow keys
    
### Move forward a page

    CTRL + f

### Move back a page

    CTRL + b

### Delete

    delete botton

### Move to bottom

    :

### Help

    :help

### Exit with no changes

    :q!

### Exit and preserve changes

    :x
